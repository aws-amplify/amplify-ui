## Usage
Import the button component and styles. 

```jsx live=true
import { Button } from "@aws-amplify/ui-react-primitives";
import "@aws-amplify/ui-react/styles.css";
// For smaller bundle size, import just the button styling:
// import "@aws-amplify/ui-react/dist/primitives/Button/styles.css";

<Button>Hello world</Button>
```

```jsx

import "@aws-amplify/ui-react/styles.css";
```

### Button sizes
Use the `size` prop to change the button size. Available options are `large`, `medium`, and `small`.

```jsx
import { Button } from "@aws-amplify/ui-react-primitives";
import "@aws-amplify/ui-react/styles.css";

<Button size="large">Large</Button>
<Button size="medium">Medium</Button>
<Button size="small">Small</Button>
```
<Button size="large">Large</Button>
<Button size="medium">Medium</Button>
<Button size="small">Small</Button>


### Button variants
Use the `variant` prop to change the button size. Available options are `large`, `medium`, and `small`.

```jsx
import { Button } from "@aws-amplify/ui-react-primitives";

<Button variant="primary">Primary</Button>
<Button variant="secondary">Secondary</Button>
<Button variant="tertiary">Tertiary</Button>
```

### CSS Styling

#### Override Amplify styling using the CSS variables:

Create a CSS file and override the CSS variables.

```css
/* styles.css */
.my-custom-button {
  --amplify-ui-button-color-bg-primary: #0057ff;
  --amplify-ui-button-color-bg-primary-hover: #4d89fc;
}
```

```jsx
import { Button } from "@aws-amplify/ui-react-primitives";
import "@aws-amplify/ui-react/styles.css";
import "./styles.css";

<Button className="my-custom-button" variant="primary">My button</Button>
```
<Button className="my-custom-button" variant="primary">My button</Button>


Or override the variables inline: 

```jsx
<div
  style={{
    "--amplify-ui-button-color-bg-primary": "#0057ff", 
    "--amplify-ui-button-color-bg-primary-hover": "#4d89fc"
  }}
  >
  <Button 
    variant="primary">
    My button
  </Button>
</div>
```
<div style={{"--amplify-ui-button-color-bg-primary": "#0057ff", "--amplify-ui-button-color-bg-primary-hover": "#4d89fc"}}>
  <Button className="my-custom-button" variant="primary">My button</Button>
</div>

#### Override styling using the classname and data attribute selectors:
Use the `amplify-ui-button` class and data attributes to style the button.


```css
/* Example: Extend all <Button> styles */
.amplify-ui-button {
  font-family: "My Custom Font", "Comic Sans MS", sans-serif;
  font-weight: bold;
}

/* Example: Override only primary variant styles */
.amplify-ui-button[data-variant=primary] {
  background-color: teal;
  color: white;
}

/* Example: Override loading styles */
.amplify-ui-button[data-loading=true] {
  opacity: 0.8;
}

/* Example: Override disabled styles */
.amplify-ui-button[disabled=true] {
  opacity: 0.8;
}

/* Example: Override button styles using Tailwind CSS */
.amplify-ui-button {
   @apply bg-blue-500 hover:bg-blue-700 text-white;
}
```
If you want to complete replace the button styling you can unset it:
```css
.amplify-ui-button {
  all: unset;
  /* Add your styling here*/
}
```

#### Styling a button instance
To override styling on a specific button: 

Using an id selector
```css
/* Example: id selector styling override */
#comic-button {
  font-family: Comic Sans, sans-serif;
}
```
```jsx
<Button id="comic-button">Click me!</Button>
```
