Messages can be dismissed by setting `isDismissible={true}` or by including `Message.Dismiss` if composing your own Message layout.

`Message.Dismiss` can be customized to hide the icon by setting the prop `showIcon={false}`. It can also accept custom button text and other props that our <Link href="/react/components/button">Button</Link> primitive accepts.
