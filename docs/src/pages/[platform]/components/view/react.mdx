## CSS Styling

### Local styling

To override styling on a specific View, you can use a class selector or style props.

_Using a class selector:_

```css
/* styles.css */
.mountain-view {
  background-color: #b7daf9;
  display: block;
  font-size: 5em;
  text-align: center;
}
````

import { View } from '@aws-amplify/ui-react';
import './styles.css';
