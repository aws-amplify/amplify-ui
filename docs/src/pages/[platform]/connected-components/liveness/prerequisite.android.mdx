> Prerequisites:
>
> 1. Install and configure the Amplify CLI by following [this guide](https://docs.amplify.aws/lib/project-setup/prereq/q/platform/android/#configuring-the-amplify-cli).
> 2. An Android application targeting at least Android SDK API level 24. For a full example of creating an Android project, please follow the [project setup walkthrough](https://docs.amplify.aws/lib/project-setup/create-application/q/platform/android/#1-create-a-new-project).
