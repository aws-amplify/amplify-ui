### Customizing Passwordless Views

Passwordless components can be styled using CSS or Amplify UI themes.

#### CSS Styling

```css
/* Passkey prompt container */
[data-amplify-authenticator-passkeyPrompt] {
  padding: 2rem;
  background: var(--amplify-colors-background-secondary);
}

/* Auth method selection */
[data-amplify-authenticator-signinselect] {
  max-width: 400px;
  margin: 0 auto;
}

/* Passkey buttons */
[data-amplify-authenticator-passkeyPrompt] button {
  font-size: 1.1rem;
  padding: 1rem 2rem;
}
```

#### Theme Customization

```tsx
import { Authenticator, ThemeProvider, Theme } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';

const theme: Theme = {
  name: 'passwordless-theme',
  tokens: {
    colors: {
      brand: {
        primary: {
          10: '#003366',
          80: '#0066cc',
          90: '#3399ff',
          100: '#66b3ff',
        },
      },
    },
    components: {
      button: {
        primary: {
          backgroundColor: { value: '{colors.brand.primary.80}' },
          _hover: {
            backgroundColor: { value: '{colors.brand.primary.90}' },
          },
        },
      },
    },
  },
};

export default function App() {
  return (
    <ThemeProvider theme={theme}>
      <Authenticator>
        {({ signOut, user }) => (
          <main>
            <h1>Hello {user?.username}</h1>
            <button onClick={signOut}>Sign out</button>
          </main>
        )}
      </Authenticator>
    </ThemeProvider>
  );
}
```

For more theming options, see the [Theming documentation](/react/theming).
