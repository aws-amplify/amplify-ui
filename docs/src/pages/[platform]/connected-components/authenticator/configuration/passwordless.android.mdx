### Passwordless

<Alert role="none" variation="info" heading="Platform Requirements">
  Passkey support requires Android API level 28+ with Google Play Services.
</Alert>

To enable passwordless authentication with passkeys, email OTP, or SMS OTP, use the `authenticationFlow` parameter:

```kotlin
Authenticator(
    authenticationFlow = AuthenticationFlow.UserChoice()
) { state ->
    Text("Welcome!")
}
```

#### Preferred Auth Factor

You can specify a preferred authentication factor to skip the factor selection step:

```kotlin
Authenticator(
    authenticationFlow = AuthenticationFlow.UserChoice(
        preferredAuthFactor = AuthFactor.WebAuthn
    )
) { state ->
    Text("Welcome!")
}
```

#### Passkey Prompts

Control when users are prompted to create a passkey:

```kotlin
Authenticator(
    authenticationFlow = AuthenticationFlow.UserChoice(
        passkeyPrompts = PasskeyPrompts(
            afterSignUp = PasskeyPrompt.Always,
            afterSignIn = PasskeyPrompt.Never
        )
    )
) { state ->
    Text("Welcome!")
}
```
