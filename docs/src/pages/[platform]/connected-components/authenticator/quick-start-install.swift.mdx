Open your project in Xcode and select **File > Add Packages...** and add the following dependencies:

- **Amplify Library for Swift**
  - Enter its GitHub URL (`https://github.com/aws-amplify/amplify-swift`), select **Up to Next Major Version** and click **Add Package**
  - Select the following libraries:
    - Amplify
    - AWSCognitoAuthPlugin

- **Amplify UI Swift - Authenticator**
  - Enter its GitHub URL (`https://github.com/aws-amplify/amplify-ui-swift-authenticator`), select **Up to Next Major Version** and click **Add Package**
  - Select the following library:
    - Authenticator
