## Tutorial

In this brief tutorial, we're going to build a basic shopping card component using [Next.js](https://nextjs.org/docs/getting-started) and Amplify UI. Then, we'll elaborate on our shopping card to play with the possibilities of using Amplify UI.

### Setup and Installation

First, execute the command below in your terminal. When prompted for the name of your project, type `amplify-ui-demo`. 

<Tabs>
<TabItem title="npm">

```shell
npx create-next-app@latest
```

</TabItem>
<TabItem title="yarn">

```shell
yarn create next-app
```

</TabItem>
</Tabs>

If you haven't already, `cd` into the `amplify-ui-demo` folder. Then, install the Amplify UI React package:

<Tabs>
<TabItem title="npm">

```shell
npm install aws-amplify @aws-amplify/ui-react
```

</TabItem>
<TabItem title="yarn">

```shell
yarn add aws-amplify @aws-amplify/ui-react
```

</TabItem>
</Tabs>
