## React and Amplify UI

Amplify UI is designed to integrate seamlessly with the React framework.

### Installation

If you haven't already, install `@aws-amplify/ui-react` with [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/):

<Tabs>
<TabItem title="npm">

```shell
npm install aws-amplify @aws-amplify/ui-react
```

</TabItem>
<TabItem title="yarn">

```shell
yarn add aws-amplify @aws-amplify/ui-react
```

</TabItem>
</Tabs>

These are the same components we use to build our connected components such as the [Authenticator](https://ui.docs.amplify.aws/react/components/authenticator). Please refer to each component's [documentation](https://ui.docs.amplify.aws/react/components/button) to see how they should be imported, configured and styled.
