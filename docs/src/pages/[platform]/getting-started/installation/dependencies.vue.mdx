For Vue 3 install `@aws-amplify/ui-vue` with [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/):

<Tabs>
<TabItem title="npm">

```shell
npm install aws-amplify @aws-amplify/ui-vue
```

</TabItem>
<TabItem title="yarn">

```shell
yarn add aws-amplify @aws-amplify/ui-vue
```

</TabItem>
</Tabs>
