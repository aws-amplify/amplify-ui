<Alert variation="info">

You will need to add dependencies for the `amplify_authenticator` to the `pubspec.yaml` file as shown below.

<Tabs justifyContent="flex-start">
  <TabItem title="Stable (Mobile)">
    <ExampleCode>
    ```yaml
      environment:
        sdk: ">=2.15.0 <3.0.0"

      dependencies:
        amplify_flutter: ^0.6.0
        amplify_auth_cognito: ^0.6.0
        amplify_authenticator: ^0.2.0
    ```
    </ExampleCode>
  </TabItem>
  <TabItem title="Developer Preview (Mobile, Web & Desktop)">
    <ExampleCode>
    ```yaml
      environment:
        sdk: ">=2.17.0 <3.0.0"
      
      dependencies:
        amplify_flutter: ^1.0.0-0
        amplify_auth_cognito: ^1.0.0-0
        amplify_authenticator: ^1.0.0-0
    ```
    </ExampleCode>
  </TabItem>
</Tabs>
