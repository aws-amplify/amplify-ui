You will need to add dependencies for the `amplify_authenticator` to the `pubspec.yaml` file as shown below.

<ExampleCode>
```yaml
  environment:
    sdk: ">=2.18.0 <4.0.0"
    flutter: ">=3.3.0"
  
  dependencies:
    amplify_flutter: ^1.0.0
    amplify_auth_cognito: ^1.0.0
    amplify_authenticator: ^1.0.0
```
</ExampleCode>
