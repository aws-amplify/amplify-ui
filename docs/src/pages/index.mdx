## Authenticator with Default Theme

Spark provides simple, clean styles to get started with a great experience in two steps:

1. `import "@aws-amplify/spark-react/styles.css";`
1. Wrap your app with `<SparkProvider theme={theme}>` to enable styles.

<Example>

```jsx
import { Authenticator, SparkProvider, theme } from "@aws-amplify/spark-react";
import "@aws-amplify/spark-react/styles.css";

<SparkProvider theme={theme}>
  <Authenticator />
</SparkProvider>;
```

## Customizing Styles

All Spark components add `data-spark-*` attributes that you can target with your CSS selectors.

In this example, `Authenticator` also has a `custom` class added so that other `Authenticator`s on the page aren't affected.

<Example>

```jsx
import { Authenticator } from "@aws-amplify/spark-react";
import "@aws-amplify/spark-react/styles.css";

<style>
  [data-spark-authenticator].custom [data-spark-label] { color: red }
</style>

<SparkProvider theme={theme}>
  <Authenticator className="custom" theme={theme} />
</SparkProvider>
```

<style>{`
  [data-spark-authenticator].custom [data-spark-label] { color: red }
`}</style>

<SparkProvider theme={theme}>
  <Authenticator className="custom" theme={theme} />
</SparkProvider>

</Example>

Of course, you can still target plain HTML tags like `button`, `a`, etc.

## Customizing Components

You can specify your own `components` to customize how Spark renders components.

In this example, three customizations are made:

1. All `Input` primitives are converted to an "underline" style
1. Only the `SignIn`'s submit button is replaced with a "3D" button
1. We replace the

```jsx
<SparkProvider
  components={{
    Input(props) {
      return (
        <input
          className="mt-0 block w-full px-0.5 border-0 border-b-2 border-gray-200 focus:ring-0 focus:border-black"
          {...props}
        />
      );
    },
    "Authenticator.SignIn.Button": (props) => {
      return props.type === "submit" ? (
        <button
          className="px-4 py-2 font-bold text-white bg-purple-500 border-b-4 border-purple-700 rounded-md shadow hover:bg-purple-600"
          {...props}
        >
          Custom Sign In
        </button>
      ) : (
        <button {...props} />
      );
    },
    "Authenticator.SignIn.PasswordControl": ({ label }) => {
      return (
        <>
          <Authenticator.SignIn.PasswordControl label={label} />
          <div className="flex items-center">
            <input className="mr-2" type="checkbox" /> Always remember this
            computer
          </div>
        </>
      );
    },
  }}
  theme={theme}
>
  <Authenticator />
</SparkProvider>
```
