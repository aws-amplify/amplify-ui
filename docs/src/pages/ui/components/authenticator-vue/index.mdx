The `Authenticator` is a simple way to add [authentication](https://docs.amplify.aws/lib/auth/getting-started/q/platform/js)
flows to your application.

## Installation

Install `@aws-amplify/ui-vue` with [npm](https://www.npmjs.com/):

```shell
npm install @aws-amplify/ui-vue --save
```

## Getting started

The simplest way to add authentication to your app is to add the `Authenticator` component:

```js{1-3,6}
<template>
  <Authenticator/>
</template>

<script setup>
import { Authenticator } from "@aws-amplify/ui-vue";
import "@aws-amplify/ui-vue/styles.css";

...

</script>
```

## Headless Usage

The `Authenticator`'s UI is powered by a state machine. You can use the `useAuth`composable
to get access to it and customizing behavior or your own UI:

```js
<template>
  <div v-if="state.matches('authenticated')">
      <h1>Welcome, {state.context.user.username}</h1>
      <button onClick={() => send("SIGN_OUT")}>Sign out</button>
  </div>
</template>

<script setup>
import { useAuth } from "@aws-amplify/ui-vue";

...

const { state, send } = useAuth();
</script>
```
