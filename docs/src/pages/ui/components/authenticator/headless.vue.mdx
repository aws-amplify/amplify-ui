Use the `user` and `signOut` slot props to customize your UI:

```html{4-6}
...
<template>
  <authenticator :loginMechanisms="loginMechanisms">
    <template v-slot="{ user, signOut }">
      <h1>Welcome, {{user.userName}}</h1>
      <button @click="signOut">Sign out</button>
    </template>
  </authenticator>
</template>
```
