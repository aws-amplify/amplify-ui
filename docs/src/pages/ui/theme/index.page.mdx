### Overview

A Theme is a structured collection of design decisions that change the appearance of a UI library. An Amplify UI theme is a structured object of typed [design tokens](https://designtokens.org).

The goals of the Amplify UI theme are:

1. **Leverage platform technologies as much as possible to be performant and support .** This means plain CSS and CSS variables. You can always fall back to writing CSS (or a pre-processer to CSS like [Sass](https://sass-lang.com/)).
1. **Use framework-specific patterns to provide an easier developer experience.**
1. **Be a good citizen**

#### Tokens

The theme structure for Amplify UI follows the [System UI Theme Specification](https://system-ui.com/theme/) with some modifications. The System UI Theme Specification outlines top-level namespaces for categorizing design token types. For example, colors go under the `colors` namespace. This specification is used by [Stitches](https://stitches.dev/docs/tokens), [Chakra-UI](https://chakra-ui.com/docs/theming/theme), and [Evergreen](https://evergreen.segment.com/introduction/theming).

### Style with theme object

The theme object is where you define tokens for color palette, font stacks, spacing, and more. Amplify UI provides a `defaultTheme` object but you can customize it to fit your design.

```jsx
import { AmplifyProvider } from '@aws-amplify/ui-react';
// Step 1: Wrap your APP with `AmplifyProvider`
const App = (
  <AmplifyProvider>
    <MyApp>/* AmplifyUI */</MyApp>
  </AmplifyProvider>
);

// Step 2: Get the theme object through useTheming hook and style components with it
import React from 'react';
import { Text, useTheme } from '@aws-amplify/ui-react';

const MyComponent = ({ children }) => {
  const { theme } = useTheme();
  return <Text color={theme.colors.font.tertiary}>{children}</Text>;
};
```

To extend or override a token in the default theme with `extendTheming`

```jsx
import { createTheme } from '@aws-amplify/ui-react';

// Step 1: Use `createTheme` and pass your custom values
const theme = createTheme({
  tokens: {
    colors: {
      font: {
        primary: { value: '#008080' },
        // ...
      },
    },
  },
});

// Step 2: Pass the new theme to `AmplifyProvider`
<AmplifyProvider theme={theme}>
  <App />
</AmplifyProvider>;

// Step 3
<Head>
  <style id="amplify" dangerouslySetInnerHTML={{ __html: theme.css }} />
</Head>;

// Step 3: Now you can use these colors in your components
const MyComponent = ({ children }) => {
  const { theme } = useTheme();
  return <Text color={theme.colors.brand.primary}>{children}</Text>;
};
```
