A Select component allows you to create a drop-down list. The `<option>` tags nested inside the component define the available options in the drop-down list.

<Example>
  <SelectDemo />
</Example>

## Usage

Import the Select primitive and styles.

```jsx
import { Select } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';

<Select defaultValue="" placeholder="Please select a fruit">
  <option value="apple">Apple</option>
  <option value="banana">Banana</option>
  <option value="orange">Orange</option>
</Select>;
```

### Controlled component

```jsx
import { useState } from 'react';
import { Select } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';

const [value, setState] = useState('');

<Select
  value={value}
  placeholder="Please select a fruit"
  onChange={(e) => setState(e.target.value)}
>
  <option value="apple">Apple</option>
  <option value="banana">Banana</option>
  <option value="orange">Orange</option>
</Select>;
```

</Example>

### Sizes

Use the `size` prop to change the Select size. Available options are `small`, `large`, and none (default).

```jsx
import { Select } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';

<Select defaultValue="" placeholder="Please select a fruit" size="small">
  <option value="apple">Apple</option>
  <option value="banana">Banana</option>
  <option value="orange">Orange</option>
</Select>;

<Select defaultValue="" placeholder="Please select a fruit">
  <option value="apple">Apple</option>
  <option value="banana">Banana</option>
  <option value="orange">Orange</option>
</Select>;

<Select defaultValue="" placeholder="Please select a fruit" size="large">
  <option value="apple">Apple</option>
  <option value="banana">Banana</option>
  <option value="orange">Orange</option>
</Select>;
```

### Variation

Select comes in 3 variants, `filled`, `flushed` and none(default). Pass the `variation` prop and set it to either of these values.

```jsx
import { Select } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';

<Select defaultValue="" placeholder="Please select a fruit">
  <option value="apple">Apple</option>
  <option value="banana">Banana</option>
  <option value="orange">Orange</option>
</Select>;

<Select defaultValue="" placeholder="Please select a fruit" variation="filled">
  <option value="apple">Apple</option>
  <option value="banana">Banana</option>
  <option value="orange">Orange</option>
</Select>;

<Select defaultValue="" placeholder="Please select a fruit" variation="flushed">
  <option value="apple">Apple</option>
  <option value="banana">Banana</option>
  <option value="orange">Orange</option>
</Select>;
```

### Icon

You can change the expand icon used in the select. Simply pass the the `icon` prop. Also, you can change the icon color by passing the `iconColor` prop.

```jsx
import { Select } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';

<Select
  defaultValue=""
  placeholder="Please select a fruit"
  icon={<IconArrowDropDown />}
  iconColor="purple"
>
  <option value="apple">Apple</option>
  <option value="banana">Banana</option>
  <option value="orange">Orange</option>
</Select>;
```

## CSS Styling

### Global styling

To override styling on all Selects, you can set the Amplify CSS variables or use the built-in `.amplify-select` class.

```css
/* styles.css */
:root {
  --amplify-components-select-background-color: yellow;
}
/* OR */
.amplify-select {
  background-color: yellow;
}
```

To replace the Select styling, unset it:

```css
.amplify-select {
  all: unset;
  /* Add your styling here*/
}
```

### Local styling

To override styling on a specific Select, you can use (in order of increasing specificity): a class selector, data attributes, or style props.

_Using a class selector:_

```css
/* styles.css */
.my-custom-select {
  background-color: cyan;
}
```

```jsx
import './styles.css';

<Select
  className="my-custom-select"
  defaultValue=""
  placeholder="Please select a fruit"
>
  <option value="apple">Apple</option>
  <option value="banana">Banana</option>
  <option value="orange">Orange</option>
</Select>;
```
