## Installation

```
npm install @aws-amplify/ui-react
```

## Usage

Import the Pagination component and styles.

```jsx
import { Pagination } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';

<Pagination
  currentPage={1}
  totalPages={10}
  siblingCount={1}
  onChange={this.onChange}
  onNext={this.onNext}
  onPrevious={this.onPrevious}
/>;
```

## CSS styling

#### Override Amplify styling using the CSS variables

<br />
Using a class and CSS file:

```css
/* styles.css */
.my-custom-pagination {
  --amplify-components-pagination-current-background-color: rgba(
    0,
    181,
    204,
    0.8
  );
  --amplify-components-pagination-button-hover-background-color: rgba(
    0,
    181,
    204,
    0.3
  );
}
```

```jsx
import { Pagination } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';
import './styles.css';

<Pagination
  currentPage={1}
  totalPages={10}
  onChange={this.onChange}
  onNext={this.onNext}
  onPrevious={this.onPrevious}
  className="my-custom-pagination"
/>;
```

<PaginationDemo isDemo={false} className="my-custom-pagination" />

<br />

Or override the variables inline:

```jsx
import { Pagination } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';

<Pagination
  currentPage={1}
  totalPages={10}
  onChange={this.onChange}
  onNext={this.onNext}
  onPrevious={this.onPrevious}
  style={{
    '--amplify-components-pagination-current-background-color':
      'rgba(219, 10, 91, 0.8)',
    '--amplify-components-pagination-button-hover-background-color':
      'rgba(219, 10, 91, 0.3)',
  }}
/>;
```

## Accessibility

By default, the root node has a role of "navigation" and each page item has an aria-label that identifies the purpose of the item ("Go to previous page", "Go to next page", "Go to page 1" etc.).
