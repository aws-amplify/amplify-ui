## Usage

Import the Placeholder component and styles.

```jsx
import { Placeholder, Flex } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';

<Flex direction="column">
  <Placeholder />
  <Placeholder />
  <Placeholder />
</Flex>
);
```

<Flex direction="column">

  <Placeholder />
  <Placeholder />
</Flex>

### There are two ways you can use the Placeholder component:

1. Pass your loading state to the `isLoaded` prop (suggested)
2. Use conditional rendering (e.g., ternary operator)

```jsx
let isLoaded = false;

<Flex direction="column">
  <Placeholder isLoaded={isLoaded}>First way</Placeholder>
  {isLoaded ? 'Second way' : <Placeholder />}
</Flex>
);
```

### Placeholder sizes

Use the `size` prop to change the placeholder size. Available options are `small` and `large`.

```jsx
import { Placeholder } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';

<Placeholder size="small" />
<Placeholder />
<Placeholder size="large" />
```

<Flex direction="column">
  <Placeholder size="small" />
  <Placeholder />
  <Placeholder size="large" />
</Flex>

### CSS Styling

Set the starting and ending colors using a custom CSS class with CSS variables:

```css
/* styles.css */
.custom-placeholder {
  --amplify-components-placeholder-start-color: purple;
  --amplify-components-placeholder-end-color: blue;
}
```

```jsx
import './styles.css';

<Placeholder className="custom-placeholder" />;
```

<Flex>
  <Placeholder className="custom-placeholder" />
</Flex>

### Override Amplify styling using props

Use style props to style an instance of `<Placeholder>`

```jsx
<Placeholder height="123px" width="50%" />
```
