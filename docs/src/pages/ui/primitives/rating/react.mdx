## Usage

Import rating component and styles

```jsx
import { Rating } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';

<Rating />;
```

<Rating />

### Rating Value

Pass in a value prop to set the rating value

```jsx
import { Rating } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';

<Rating value={3.7} />;
```

<Rating value={3.7} />

### Max Value

Pass in a maxValue prop to set the maximum value of the rating component

```jsx
import { Rating } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';

<Rating value={6.3} maxValue={10} />;
```

<Rating value={6.3} maxValue={10} />

### Size

Use the size property to adjust the rating component the default size is medium, other options include small and large

```jsx
import { Rating } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';

<Rating value={2.5} size="small" />;
<Rating value={2.5} />;
<Rating value={2.5} size="large" />;
```

<Rating value={2.5} size="small" />
<Rating value={2.5} />
<Rating value={2.5} size="large" />

### Colors

The fill color and empty color can be set using the `fillColor` and `emptyColor` props

```jsx
import { Rating } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';

<Rating value={2.5} fillColor="red" emptyColor="green" />;
```

<Rating value={2.5} fillColor="red" emptyColor="green" />

### Custom Icon

A custom icon can be used in the rating component using the icon prop. The rating component works well with all major icon sets and these icons can be substituted in using this icon prop. The icon prop will override both the filled and empty icons unless a specific empty icon is also passed in.

```jsx
import { Rating, IconAdd } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';

<Rating value={2.5} icon={<IconAdd />} />;
```

<Rating value={2.5} icon={<IconAdd />} />

### Custom Empty Icon

A custom empty icon can be used in the rating component using the emptyIcon prop. The emptyIcon prop will only effect the empty icon being rendered and by passing in an empty icon there will be different icons being rendered for the empty icons and filled icons. This is in contrast to the icon prop which will override both the empty and filled icons being used.

```jsx
import { Rating, IconStarBorder } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';

<Rating value={2.5} emptyIcon={<IconStarBorder />} />;
```

<Rating value={2.5} emptyIcon={<IconStarBorder />} />

## CSS Styling

### component level overrides

The rating component styling props can be overridden using a custom class name and the built-in CSS classes.

#### Filled and empty icon colors

The empty and filled icon colors can be overridden using a custom CSS class on your rating component and with the `.amplify-rating-icon-filled` and `.amplify-rating-icon-empty` classes

```css
.my-rating-component .amplify-rating-icon-filled {
  color: purple;
}

.my-rating-component .amplify-rating-icon-empty {
  color: black;
}
```

```jsx
import { Rating } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';

<Rating value={2.5} className="my-rating-component" />;
```

<Rating value={2.5} className="my-rating-component" />

#### Sizes

The size variations on the rating component can be overridden at the individual component level

```css
.large-rating[data-size='small'] {
  font-size: 40px;
  line-height: 40px;
}
```

```jsx
import { Rating } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';

<Rating value={2.5} className="large-rating" size="small" />;
```

<Rating value={2.5} className="large-rating" size="small" />

### Application wide styling

All rating components within an application can be styled using CSS variables at the root level

#### Filled and empty icon colors

The empty and filled icon colors can be overridden using css variables

```css
:root {
  --amplify-components-rating-filled-color: purple;
  --amplify-components-rating-empty-color: black;
}
```

```jsx
import { Rating } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';

<Rating value={2.5} className="application-color-override" />;
```

<Rating value={2.5} className="application-color-override" />

#### Sizes

The size variations on the rating component can be overridden at the application level using css variables

```css
:root {
  --amplify-components-rating-small-size: 40px;
}
```

```jsx
import { Rating } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';

<Rating value={2.5} size="small" />;
```

<Rating value={2.5} className="application-size-override" size="small" />
