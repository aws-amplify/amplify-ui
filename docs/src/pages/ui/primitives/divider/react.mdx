## Usage

Import the Divider component and styles.

```jsx
import { Divider } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';

<Divider />;
```

<Example>
  <Divider />
</Example>

### Orientation

**Horizontal (default)**

```jsx
<View>
  <Text>Before</Text>
  <Divider orientation="horizontal" />
  <Text>After</Text>
</View>
```

<Example>
  <View>
    <Text>Before</Text>
    <Divider orientation="horizontal" />
    <Text>After</Text>
  </View>
</Example>

**Vertical**

```jsx
<Flex height="100px" justifyContent="space-around">
  <Text>Before</Text>
  <Divider orientation="vertical" />
  <Text>After</Text>
</Flex>
```

<Example>
  <Flex height="100px" justifyContent="space-around">
    <Text>Before</Text>

    <Text>After</Text>
  </Flex>
</Example>

## CSS Styling

### Global styling

To override styling on all Dividers, you can set the Amplify CSS variables or use the built-in `.amplify-divider` class.

```css
/* styles.css */
:root {
  --amplify-components-divider-border-style: dashed;
}
/* OR */
.amplify-divider {
  border-style: dashed;
}
```

<Example>
  <Divider className="border-style-override" />
</Example>

To replace the Divider styling, unset it:

```css
.amplify-divider {
  all: unset;
  /* Add your styling here*/
}
```
