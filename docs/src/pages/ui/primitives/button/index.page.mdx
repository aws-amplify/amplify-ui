## Usage

Import the button component and styles.

```jsx
import { Button } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';
// For smaller bundle size, import just the button styling:
// import "@aws-amplify/ui-react/dist/primitives/Button/styles.css";

<Button>Hello world</Button>;
```

<Button>Hello world</Button>

```jsx

import '@aws-amplify/ui-react/styles.css';
```

### Button sizes

Use the `size` prop to change the button size. Available options are `large`, `medium`, and `small`.

```jsx
import { Button } from "@aws-amplify/ui-react";
import "@aws-amplify/ui-react/styles.css";

<Button size="large">Large</Button>
<Button size="medium">Medium</Button>
<Button size="small">Small</Button>
```

<Button size="large">Large</Button>
<Button size="medium">Medium</Button>
<Button size="small">Small</Button>

### Button variations

Use the `variation` prop to change the button variation. Available options are `primary`, `secondary`, `tertiary` and `link`.

```jsx
import { Button } from "@aws-amplify/ui-react";
import "@aws-amplify/ui-react/styles.css";

<Button variation="primary">Primary</Button>
<Button>Default</Button>
<Button variation="link">Link</Button>
```

<Button variation="primary">Primary</Button>
<Button>Default</Button>
<Button variation="link">Link</Button>

### CSS Styling

#### Override Amplify styling using the CSS variables

Using a class and CSS file:

```css
/* styles.css */
.my-custom-button {
  --amplify-ui-components-button-primary-background-color: #0057ff;
  --amplify-ui-components-button-primary-hover-background-color: #4d89fc;
}
```

```jsx
import { Button } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';
import './styles.css';

<Button className="my-custom-button" variation="primary">
  My button
</Button>;
```

<Button className="my-custom-button" variation="primary">
  My button
</Button>

Or override the variables inline:

```jsx
<div
  style={{
    '--amplify-ui-components-button-primary-background-color': '#0057ff',
    '--amplify-ui-components-button-primary-hover-background-color': '#4d89fc',
  }}
>
  <Button variation="primary">My button</Button>
</div>
```

<div
  style={{
    '--amplify-ui-components-button-primary-background-color': '#0057ff',
    '--amplify-ui-components-button-primary-hover-background-color': '#4d89fc',
  }}
>
  <Button variation="primary">My button</Button>
</div>

#### Override styling using the classname and data attribute selectors:

Use the `amplify-ui-button` class and data attributes to style all buttons.

```css
/* Example: Extend all <Button> styles */
.amplify-ui-button {
  font-family: 'My Custom Font', 'Comic Sans MS', sans-serif;
  font-weight: bold;
}

/* Example: Override only primary variation styles */
.amplify-ui-button[data-variation='primary'] {
  background-color: teal;
  color: white;
}

/* Example: Override loading styles */
.amplify-ui-button[data-loading='true'] {
  opacity: 0.8;
}

/* Example: Override disabled styles */
.amplify-ui-button[disabled='true'] {
  opacity: 0.8;
}

/* Example: Override button styles using Tailwind CSS */
.amplify-ui-button {
  @apply bg-blue-500 hover:bg-blue-700 text-white;
}
```

To replace the button styling, unset it:

```css
.amplify-ui-button {
  all: unset;
  /* Add your styling here*/
}
```

#### Styling a button instance

To override styling on a specific button

_Using an class selector:_

```css
/* Example: class selector styling override */
.colorful-button {
  background: linear-gradient(90deg, #fdbb2d 0%, #22c1c3 100%);
}
```

```jsx
<Button className="colorful-button">I'm colorful!</Button>
```

<Button className="colorful-button">I'm colorful!</Button>

_Using an id selector:_

```css
/* Example: id selector styling override */
#gradient-button {
  color: white;
  border-radius: 2rem;
  background: rgb(2, 0, 36);
  background: linear-gradient(
    90deg,
    rgba(2, 0, 36, 1) 0%,
    rgba(9, 9, 121, 1) 35%,
    rgba(0, 212, 255, 1) 100%
  );
}
```

```jsx
<Button id="gradient-button">I'm beautiful!</Button>
```

<Button id="gradient-button">I'm beautiful!</Button>
