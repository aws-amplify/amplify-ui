For Vue 3 install `@aws-amplify/ui-vue@next` with [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/):

<BlockSwitcher>
<Block name="npm">

```shell
npm install aws-amplify @aws-amplify/ui-vue@next
```

</Block>
<Block name="yarn">

```shell
yarn add @aws-amplify/ui-vue@next
```

</Block>
</BlockSwitcher>

**For Vue 2 please follow the installation instructions listed [here.](https://docs.amplify.aws/ui/q/framework/vue/)**
