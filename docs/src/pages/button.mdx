# Button primitive

## Installation

```
npm install @aws-amplify/ui-react
```

## Usage
Import the button component and styles. 

```jsx live=true
import { ButtonElement } from "@aws-amplify/ui-react";
import "@aws-amplify/ui-react/styles.css";
// For smaller bundle size, import just the button styling:
import "@aws-amplify/ui-react/dist/primitives/ButtonElement/styles.css";

<ButtonElement>Click me</ButtonElement>
```
<ButtonElement>Hello world</ButtonElement>

### Button sizes
Use the `size` prop to change the button size. Available options are `large`, `medium`, and `small`.

```jsx
import { ButtonElement } from "@aws-amplify/ui-react";
import "@aws-amplify/ui-react/styles.css";

<ButtonElement size="large">Large</ButtonElement>
<ButtonElement size="medium">Medium</ButtonElement>
<ButtonElement size="small">Small</ButtonElement>
```
<ButtonElement size="large">Large</ButtonElement>
<ButtonElement size="medium">Medium</ButtonElement>
<ButtonElement size="small">Small</ButtonElement>

Use the `variant` prop to change the button size. Available options are `large`, `medium`, and `small`.

## Implementation comparison:

Both implementations share the same external API and Theme Design Tokens. The primary difference between the implementations is in implementation and limitations around use.

### Chakra Button

Pros:

- Custom Theming is user friendly and easy to learn
- Works well with Style Dictionary (could generate the theme directly)
-

Cons:

- Must be used used with the AmpliyUIProvider which is a wrapper around the ChakraProvider.
- You cannot nest ChakraProviders, so if our customers are already using Chakra in their App, they will have to extend their theme with ours (we would provide helper methods).
- Customers wouldn't be able to use our Amplify Chakra theme by itself unless we provide styling for ALL Chakra components.
- Bundle size: If customers are not using Chakra, they have to bring all of Chakra in to benefit to use our components

Raw Button

```js
export interface ButtonProps extends BaseComponentProps {
  /**
   * If `true`, the button will show a spinner.
   */
  loading?: boolean;

  /**
   * If `true`, the button will be styled in its active state.
   */
  active?: boolean;

  /**
   * If `true`, the button will be disabled.
   */
  disabled?: boolean;

  /**
   * The label to show in the button when `loading` is true
   * If no text is passed, it only shows the spinner
   */
  loadingText?: string;

  /**
   * If `true`, the button will take up the full width of its container.
   */
  fullWidth?: boolean;

  /**
   * Button click event handler
   */
  onClick?: (event: React.MouseEvent) => void;

  /**
   * The html button type to use.
   */
  type?: ButtonTypes;

  /**
   * Changes the size of the button.
   * @default "medium"
   */
  size?: "small" | "medium" | "large";

  /**
   * Changes the visual weight of the button.
   * @default "secondary"
   */
  variant?: "primary" | "secondary" | "tertiary" | "link";

  /**
   * Aria label for accessibility
   * @default undefined
   */
  ariaLabel?: string;
}
```
