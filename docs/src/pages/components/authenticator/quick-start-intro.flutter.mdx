> Prerequisite: [Create an Authentication backend](https://docs.amplify.aws/lib/auth/getting-started/q/platform/flutter) and add the [amplify_authenticator](https://pub.dev/packages/amplify_authenticator) package as a dependency.

The quickest way to get started is by wrapping your MaterialApp widget with the Authenticator widget.
Once an end-user has created an account & signed in, the MaterialApp will be displayed.
