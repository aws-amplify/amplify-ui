## Usage

Import the Placeholder component and styles.

```jsx
import { Placeholder, Flex } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';

<Flex direction="column">
  <Placeholder />
  <Placeholder />
  <Placeholder />
</Flex>
);
```

<Flex direction="column">
  <Placeholder />
  <Placeholder />
  <Placeholder />
</Flex>

There are two ways you can use the Placeholder component:

1. Pass your loading state to the `isLoaded` prop (suggested)
2. Use conditional rendering (e.g., ternary operator)

```jsx
let isLoaded = false;

<Flex direction="column">
  <Placeholder isLoaded={isLoaded}>First way</Placeholder>
  {isLoaded ? 'Second way' : <Placeholder />}
</Flex>
);
```

### Sizes

Use the `size` prop to change the Placeholder size. Available options are `small`, `large`, and none (default).

```jsx
<Placeholder size="small" />
<Placeholder />
<Placeholder size="large" />
```

### Global styling

To override styling on all Placeholder components, you can set the Amplify CSS variables or use the built-in `.amplify-placeholder` class.

(Example needed)

To replace the Placeholder styling, unset it:

```css
.amplify-placeholder {
  all: unset;
  /* Add your styling here*/
}
```

Set the starting and ending colors for the Placeholder animation using a custom CSS class with the Amplify CSS variables:

```css
/* styles.css */
.custom-placeholder {
  --amplify-components-placeholder-start-color: purple;
  --amplify-components-placeholder-end-color: blue;
}
```

```jsx
import './styles.css';

<Placeholder className="custom-placeholder" />;
```

<Flex>
  <Placeholder className="custom-placeholder" />
</Flex>
