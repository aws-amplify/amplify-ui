## CSS Styling

### Target classes

<ComponentStyleDisplay componentName="Placeholder" />

### Global styling

To override styling on all Placeholder components, you can set the Amplify CSS variables or use the built-in `.amplify-placeholder` class.

<Example>
    <Placeholder height="2rem" borderRadius="1rem" />
  
  <ExampleCode>
    ```css
    /* styles.css */
    :root {
      --amplify-components-placeholder-height: 2rem;
      --amplify-components-placeholder-border-radius: 1rem;
    }
    /* OR */
    .amplify-placeholder {
      height: 2rem;
      border-radius: 1rem;
    }
    ```

  </ExampleCode>
</Example>

### Local styling

To override styling on a specific Placeholder, you can use a class selector or style props.

_Using a class selector:_

Set the starting and ending colors for the Placeholder animation using a custom CSS class with the Amplify CSS variables:

<Example>
   <Placeholder className="custom-placeholder" />
  <ExampleCode>
    ```css
    /* styles.css */
    .custom-placeholder {
      --amplify-components-placeholder-start-color: purple;
      --amplify-components-placeholder-end-color: blue;
    }
    ```
  </ExampleCode>
  <ExampleCode>
    ```jsx
    import './styles.css';

    <Placeholder className="custom-placeholder" />
    ```

  </ExampleCode>
</Example>

_Using style props:_

<Example>
  <Placeholder height="123px" width="50%" />
  <ExampleCode>


  </ExampleCode>
</Example>
