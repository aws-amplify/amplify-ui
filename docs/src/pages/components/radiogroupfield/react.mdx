A RadioGroupField allows users to select a single option from a list of mutually exclusive options.

<Example>
  <Demo />
</Example>

## Usage

Import the Radio, RadioGroupField and styles.

```jsx
import { Radio, RadioGroupField } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';

<RadioGroupField label="Language" name="language" defaultValue="html">
  <Radio value="html">html</Radio>
  <Radio value="css">css</Radio>
  <Radio value="javascript">javascript</Radio>
</RadioGroupField>;
```

<Example>
  <RadioGroupField
    label="Language"
    name="example-1"
    defaultValue="html"
    labelHidden
  >
    <Radio value="html">html</Radio>
    <Radio value="css">css</Radio>
    <Radio value="javascript">javascript</Radio>
  </RadioGroupField>
</Example>

### Controlled component

```jsx
import * as React from 'react';
import { RadioGroupField, Radio } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';

const [value, setValue] = React.useState('html');

<RadioGroupField
  label="Language"
  name="language"
  value={value}
  onChange={(e) => setValue(e.target.value)}
>
  <Radio value="html">html</Radio>
  <Radio value="css">css</Radio>
  <Radio value="javascript">javascript</Radio>
</RadioGroupField>;
```

<Example>
  <ControlledRadioGroupField />
</Example>

```jsx
  <Radio value="html">html</Radio>
</RadioGroupField>;
```

<RadioGroupField

### State

The available RadioGroupField states include `isDisabled` and `isReadonly`. A disabled field will be not be focusable not mutable and will not be submitted with form data. A readonly field cannot be edited by the user.

#### Disabled radios and radio group

```jsx
import { Radio, RadioGroupField } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';

// disabled a single radio
<RadioGroupField label="Language" name="disabled-single" defaultValue="html">
  <Radio value="html">html</Radio>
  <Radio value="css" isDisabled={true}>
    css
  </Radio>
  <Radio value="javascript">javascript</Radio>
</RadioGroupField>;

// disabled the whole group
<RadioGroupField
  label="Language"
  name="disabled-group"
  defaultValue="html"
  isDisabled={true}
>
  <Radio value="html">html</Radio>
  <Radio value="css">css</Radio>
  <Radio value="javascript">javascript</Radio>
</RadioGroupField>;
```

### Validation error

Use the `hasError` and `errorMessage` props to mark a RadioGroupField as having an validation error.

```jsx
import { Radio, RadioGroupField } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';

// Use the `hasError` and `errorMessage` fields
<RadioGroupField
  label="Language"
  name="language"
  errorMessage="This is a required field."
  hasError
>
  <Radio value="html">html</Radio>
  <Radio value="css">css</Radio>
  <Radio value="javascript">javascript</Radio>
```

## CSS Styling

### Global styling

To override styling on all Radio buttons, you can set the Amplify CSS variables or use the built-in `.amplify-radio__button` class.

```css
/* styles.css */
:root {
  --amplify-components-radio-button-border-color: yellow;
  --amplify-components-radio-button-background-color: yellow;
}
/* OR */
.amplify-radio__button {
  border-color: yellow;
  background-color: yellow;
}
```

To replace Radio button styling, unset it:

```css
.amplify-radio__button {
  all: unset;
  /* Add your styling here*/
}
```

### Local styling

To override styling on a specific Radio, you can use (in order of increasing specificity): a class selector, data attributes, or style props.

_Using a class selector:_

```css
/* styles.css */
.custom-radio .amplify-radio__input:checked + .amplify-radio__button {
  color: white;
  border-color: crimson;
  background-color: crimson;
}
```

```jsx
import { Radio, RadioGroupField } from '@aws-amplify/ui-react';
import '@aws-amplify/ui-react/styles.css';

<RadioGroupField label="Language" name="language" defaultValue="html">
  <Radio value="html" className="custom-radio">
    html
  </Radio>
</RadioGroupField>;
```
