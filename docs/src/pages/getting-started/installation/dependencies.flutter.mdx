Add the `amplify_authenticator` package with [pub](https://pub.dev/):

```shell
flutter pub add amplify_authenticator
```

This will add a dependency like this to your package's pubspec.yaml

```yaml
dependencies:
  amplify_authenticator: ^0.1.0
```
